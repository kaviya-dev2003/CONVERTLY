Problem: Lead collection platforms often lack structured frontend-backend integration and mobile optimization.

Solution: Developed Convertly, a full-stack lead management SaaS with secure contact form handling, responsive UI, and modular architecture.

Key Features:
- React frontend with responsive layouts
- Node.js + Express backend for data handling
- Secure MySQL database
- Glassmorphism & modern UI design

Tech Stack: React.js, Node.js, Express.js, MySQL
